# Seaborn·Plotly 시각화 결과 및 종합 의견

## 1. 시각화 목적

UCI Adult 성인 소득 데이터를 이용하여 인구통계 특성과 고소득 여부의 관계를 확인하였다. 정적 차트는 Seaborn으로 소득 집단별 연령 분포를 비교하고, 인터랙티브 차트는 Plotly로 학력과 성별에 따른 고소득(`>50K`) 비율을 비교하였다.

- 입력 데이터: `data/adult.data`
- Seaborn 결과: `outputs/seaborn_age_distribution.png`
- Plotly 결과: `outputs/plotly_income_by_education_gender.html`

원본 32,561행에서 결측값과 중복값을 제거한 뒤 30,139행을 시각화에 사용하였다. 정제된 데이터의 고소득자는 7,506명이며, 전체의 약 24.9%였다.

## 실행 환경 및 패키지 설치

### 실행 환경

- Python 3.9 이상
- 입력 데이터: `data/adult.data`

### 가상환경 생성 및 활성화

프로젝트 폴더에서 다음 명령을 실행한다.

```bash
python3 -m venv .venv
source .venv/bin/activate
```

터미널 프롬프트 앞에 `(.venv)`가 표시되면 가상환경이 정상적으로 활성화된 것이다.

### 필수 패키지 설치

```bash
python -m pip install --upgrade pip
python -m pip install pandas matplotlib seaborn plotly
```

각 패키지의 역할은 다음과 같다.

- `pandas`: 데이터 불러오기, 정제 및 그룹별 집계
- `matplotlib`: 정적 차트 생성 및 이미지 저장
- `seaborn`: 소득 집단별 연령 분포 시각화
- `plotly`: 학력·성별 고소득 비율 인터랙티브 차트 생성

`pathlib`은 Python 기본 라이브러리이므로 별도로 설치하지 않는다.

### 프로그램 실행

정상적으로 실행되면 다음 결과가 생성된다.

- `outputs/seaborn_age_distribution.png`
- `outputs/plotly_income_by_education_gender.html`

Plotly 차트는 다음 명령으로 브라우저에서 확인할 수 있다.

```bash
open outputs/plotly_income_by_education_gender.html
```

> `.venv/` 폴더는 개인 실행 환경이므로 Git 커밋에 포함하지 않는다.

## 2. 코드 구성 및 처리 과정

### 데이터 불러오기와 정제

`load_and_clean_data()`는 헤더가 없는 Adult 데이터에 15개 컬럼명을 부여하고 데이터를 불러온다. `?`를 결측값으로 처리하고, 문자열의 앞뒤 공백과 소득 값 끝의 마침표를 제거한다. 이후 결측 행과 중복 행을 제거하고 다음과 같은 파생 변수를 만든다.

- `income_label`: 차트 범례에 사용할 `<=50K`, `>50K` 문자열
- `is_high_income`: 고소득이면 1, 아니면 0인 이진 변수

`is_high_income`의 그룹별 평균은 해당 그룹의 고소득자 비율과 같기 때문에 Plotly 막대 차트의 y축 값으로 사용할 수 있다.

### Seaborn 정적 차트

`create_seaborn_chart()`는 `sns.histplot()`으로 저소득 집단과 고소득 집단의 연령 분포를 겹쳐서 표현한다. 히스토그램과 KDE 곡선을 함께 사용하여 연령대별 빈도뿐 아니라 전체적인 분포 형태도 확인할 수 있다.

`stat="density"`와 `common_norm=False`를 적용했으므로 두 집단의 크기가 다르더라도 각 집단 내부의 분포 형태를 비교하기 쉽다. 다만 이 차트의 높이만으로 두 집단의 실제 인원수를 직접 비교해서는 안 된다.

### Plotly 인터랙티브 차트

`create_plotly_chart()`는 학력과 성별로 데이터를 그룹화하고 고소득 비율과 표본 수를 계산한다. `px.bar()`의 그룹형 막대 차트를 사용하여 같은 학력 안에서 성별 차이를 나란히 비교할 수 있다.

마우스를 막대 위에 올리면 학력, 고소득 비율, 표본 수가 표시된다. 범례 선택, 확대 및 축소 기능을 이용할 수 있어 정적 이미지보다 세부 그룹을 탐색하기 편리하다. 학력 순서는 전체 고소득 비율이 높은 순서로 정렬하였다.

## 3. 주요 결과

### 연령 분포 결과

- 저소득 집단의 평균 연령은 약 36.6세이고 중앙값은 34세였다.
- 고소득 집단의 평균 연령은 약 44.0세이고 중앙값은 43세였다.
- 두 집단의 평균 연령 차이는 약 7.3세로, 고소득 집단의 분포가 상대적으로 높은 연령대에 형성되었다.
- 저소득 집단의 연령 표준편차는 약 13.5세, 고소득 집단은 약 10.3세로 나타나 저소득 집단의 연령 분포가 더 넓었다.

이 결과는 해당 데이터에서 연령과 고소득 여부가 서로 관련된 패턴을 보인다는 뜻이다. 그러나 시각화만으로 연령 증가가 고소득의 직접적인 원인이라고 결론 내릴 수는 없다.

### 학력·성별 그룹 비교 결과

전체 성별 고소득 비율은 남성 약 31.4%, 여성 약 11.4%로 나타났다. 학력별 전체 고소득 비율이 높은 그룹은 다음과 같다.

| 학력 | 표본 수 | 고소득 비율 |
|---|---:|---:|
| Prof-school | 542명 | 74.9% |
| Doctorate | 375명 | 74.7% |
| Masters | 1,626명 | 56.5% |
| Bachelors | 5,042명 | 42.2% |
| Assoc-voc | 1,307명 | 26.3% |

전문대학원, 박사, 석사, 학사 순으로 고소득 비율이 높게 나타나 전반적으로 교육 수준과 고소득 비율 사이에 양의 관련성이 확인되었다. 같은 학력 안에서도 성별에 따른 차이가 관찰되었다. 예를 들어 석사 그룹의 고소득 비율은 남성 약 66.8%, 여성 약 33.8%였고, 학사 그룹은 남성 약 51.4%, 여성 약 20.8%였다.

단, 그룹별 표본 수가 서로 다르며 연령, 직업, 주당 근로시간, 혼인 상태 등의 변수를 동시에 통제하지 않았다. 따라서 이 결과 역시 인과관계나 차별 여부를 직접 증명하는 결과가 아니라 데이터에 나타난 집단별 연관성으로 해석해야 한다.

## 4. 결과 종합(공통)

두 시각화를 함께 보면 고소득 여부는 하나의 특성만으로 설명되기보다 연령, 학력, 성별 등 여러 특성과 복합적으로 관련되어 있음을 알 수 있다. Seaborn 차트에서는 고소득 집단이 상대적으로 높은 연령대에 분포했고, Plotly 차트에서는 교육 수준이 높은 그룹일수록 고소득 비율이 높아지는 경향과 동일 학력 내 성별 차이가 확인되었다.

정적 차트는 보고서나 발표 자료에서 전체 패턴을 빠르게 전달하는 데 적합하고, 인터랙티브 차트는 사용자가 관심 있는 학력과 성별 그룹을 직접 선택하여 세부 값을 확인하는 데 적합했다. 두 방식을 함께 사용함으로써 결과 전달력과 탐색 가능성을 모두 확보하였다.

## 5. 팀원 의견

- Seaborn 차트는 소득 집단별 연령 분포가 한 화면에 제시되어 두 집단의 중심 위치와 분포 형태를 직관적으로 비교할 수 있다는 장점이 있다.
- Plotly 차트는 마우스 호버로 비율과 표본 수를 동시에 확인할 수 있어 단순 막대 높이만 비교할 때 발생할 수 있는 해석 오류를 줄이는 데 도움이 된다.
- 학력별 막대가 많아 작은 화면에서는 x축 라벨이 복잡해 보일 수 있으므로 발표에서는 화면을 넓게 사용하거나 상위 학력 그룹을 별도로 강조하는 것이 좋다.
- 집단별 비율 차이를 공정하게 비교하려면 표본 수뿐 아니라 신뢰구간이나 통계적 유의성도 함께 제시할 필요가 있다.
- 민감한 인구통계 변수인 성별을 다루므로 결과를 개인의 능력 차이로 해석하지 않고, 데이터가 수집된 시기와 사회·직업 구조의 영향을 함께 고려해야 한다.

## 6. 코드 품질 측면의 장점

- 데이터 적재, 정제, Seaborn 시각화, Plotly 시각화가 각각 함수로 분리되어 역할이 명확하다.
- `pathlib.Path`를 사용하여 운영체제에 따른 경로 차이를 줄였다.
- 입력 데이터가 없거나 정제 후 데이터가 비었을 때 예외를 발생시켜 원인을 쉽게 확인할 수 있다.
- 출력 폴더가 없으면 자동 생성하며, Matplotlib figure를 `plt.close()`로 닫아 메모리 누수를 방지한다.
- 여러 운영체제의 한글 폰트를 후보로 확인하여 차트의 한글 깨짐 가능성을 낮췄다.
- Plotly HTML에 필요한 JavaScript를 포함하므로 생성된 파일을 인터넷 연결 없이도 열 수 있다.
- 함수 설명, 변수명, 타입 힌트가 포함되어 코드의 목적과 데이터 흐름을 이해하기 쉽다.

## 7. 개선 사항

### 우선 적용할 개선

1. **중복 폰트 호출 제거**  
   현재 `create_seaborn_chart()`에서 `set_korean_font()`가 두 번 호출된다. `sns.set_theme()`가 폰트 설정을 초기화할 수 있으므로 첫 번째 호출은 제거하고 아래 순서만 유지하는 것이 좋다.

   ```python
   sns.set_theme(style="whitegrid")
   set_korean_font()
   ```

2. **소득 값 검증 강화**  
   현재 `>50K`가 아닌 모든 값을 `<=50K`로 처리한다. 예상하지 못한 값도 저소득으로 분류될 수 있으므로 허용값을 확인하는 검증 코드를 추가하는 것이 안전하다.

   ```python
   valid_income = {"<=50K", ">50K"}
   unknown_income = set(df["income"].dropna().unique()) - valid_income
   if unknown_income:
       raise ValueError(f"알 수 없는 소득 값: {unknown_income}")
   ```

3. **결측값 제거 영향 기록**  
   현재 하나의 값이라도 누락된 행을 모두 제거한다. 제거 전후 행 수와 컬럼별 결측 수를 출력하고, 필요하면 범주형 변수에 `Unknown`을 적용하는 방식도 비교할 필요가 있다. 결측 제거가 특정 집단을 더 많이 제외하면 결과에 편향이 생길 수 있다.

4. **불확실성 표현 추가**  
   그룹별 고소득 비율에 95% 신뢰구간을 추가하면 막대 높이 차이가 표본 오차 범위보다 큰지 판단하기 쉽다. 표본 수가 너무 적은 그룹에는 별도 표시를 하거나 최소 표본 기준을 설정하는 방법도 고려할 수 있다.

### 추가 개선 아이디어

- Seaborn 차트에 집단별 평균선과 표본 수를 표시하면 결과 해석이 더 쉬워진다.
- 파일 경로, 차트 제목, 출력 형식을 명령행 인자로 받으면 다른 데이터에도 재사용하기 쉽다.
- Plotly 파일 크기가 중요한 경우 `include_plotlyjs="cdn"`을 사용할 수 있지만, 이 경우 차트를 열 때 인터넷 연결이 필요하다는 단점이 있다.
- 데이터 정제 및 그룹 집계 함수에 단위 테스트를 추가하면 소득 값 변환과 비율 계산 오류를 조기에 발견할 수 있다.
- 린터와 포매터를 적용하면 팀원이 작성한 다른 코드와 스타일을 통일하기 쉽다.

## 8. 종합 의견

이번 시각화 코드는 과제에서 요구한 Seaborn 정적 차트와 Plotly 인터랙티브 차트를 각각 구현했으며, 분포 비교와 그룹 비교라는 서로 다른 분석 목적을 충족한다. 데이터 적재부터 결과 저장까지 한 파일에서 자동으로 수행되고 함수별 책임도 비교적 명확하여 재실행성과 가독성이 좋다.

분석 결과에서는 고소득 집단의 연령대가 더 높고 교육 수준이 높은 집단의 고소득 비율이 전반적으로 높았으며, 같은 학력 안에서도 성별 차이가 관찰되었다. 다만 이러한 차트는 변수 간 연관성을 보여주는 탐색적 분석이므로 인과관계로 확대 해석해서는 안 된다. 향후 결측 처리 영향, 그룹별 신뢰구간, 다른 변수의 통제 및 데이터 수집 시점의 한계를 함께 검토하면 분석의 신뢰도를 더 높일 수 있다.
